# Running Insights

ランニング中の気付き、感想、工夫、発見などを記録するためのリポジトリです。

## ディレクトリ構成

- `TEMPLATE.md` : 記録用のテンプレートファイル
- `logs/` : 日付ごとの記録を保存するディレクトリ

## 使い方

1. `TEMPLATE.md` をコピーして、新しいファイルを作成します。
   - ファイル名の例: `logs/YYYY-MM-DD.md`
2. テンプレートに沿って、ランニングの記録と気付きを記入します。
3. リポジトリにコミットしてプッシュします。

## 記録する内容の例

- フォームに関する気付き（例：腕の振りを意識したら楽になった）
- ペース配分についての反省
- 走っている最中に思いついたアイデア
- 季節や天候による体調の変化
- 新しいランニングコースの発見
